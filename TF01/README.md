# Servidor de site estatico com Nginx

## Passos para rodar o App
Tenha WSL Ubuntu em sua máquina local e após acessar 

## Acessando Ubuntu:
Rode no terminal (Powershell ou CMD):

```sh
wsl -d ubuntu
```

## Acessando a Home:
acesse o diretório padrão do ubuntu:

```sh
cd $HOME
```

## Clonando Repositório: 
Rode este comando em seu terminal bash:

```sh
git clone https://github.com/Viniciusgigante19/TF-deploy-with-nginx.git
```

## Acessando diretório do repositório:
Entre no diretório do repositório usando:

```sh
cd ./TF-deploy-with-nginx
```

## Concedendo Permissão:
Use este comando para permitir que scripts rode com permissão:

```sh
chmod +x ./TF01/scripts/install.sh
```

## Iniciando projeto:
Rode este script usando este comando para iniciar o projeto:

```sh
sudo ./TF01/scripts/install.sh
```
